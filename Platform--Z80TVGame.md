# Hardware summary

* Z80 @ 4 Mhz
* 16k RAM, 8k VRAM
* 32k cartridge supports
* Single joystick
* 1 bit sound
* Custom B&W graphics 

# Compilation

    zcc +z80tvgame file.c -create-app

A .rom file is created suitable for loading into an emulator

# Background

This target is a home-brew console designed and built by Mr Isizu.
